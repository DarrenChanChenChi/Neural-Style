# 照片融合 #

这是Torch的一个简单示例，其利用卷积神经网络的理论，参照论文A Neural Algorithm of Artistic Style，可以实现一种效果：两张图片，一张取其内容，另一张取其风格，然后将两张融合到一起。
本代码是服务端，配好环境后，通过本代码，可以直接上传照片，调用算法，返回结果。
